    <h2 id="installation">ğŸ› ï¸ Installation Steps</h2>
    <h3>1. Clone the repository</h3>
    <pre><code>git clone git@github.com:Devansh501/Linktree-app.git
cd Linktree</code></pre>
    <h3>2. Install Dependencies</h3>
    <pre><code># This installs dependencies for the root, frontend, and backend
npm install</code></pre>
    <h3>3. Environment Setup</h3>
    <p>Create <code>.env</code> files in both the <code>server</code> and <code>site</code> folders with your specific configurations (MongoDB URI, JWT Secret, etc.).</p>
    <h3>4. Run the Project</h3>
    <p><strong>Start Backend:</strong></p>
    <pre><code>cd server
npm start</code></pre>
    <p><strong>Start Frontend:</strong></p>
    <pre><code>cd site
npm run dev</code></pre>
